### **Usage**

```bash
  git clone https://github.com/MrMohammedMath/profile-card-component.git

  cd profile-card-component

  pnpm install
```

- for running dev server

```bash
pnpm run start
```

- for build & review

```bash
pnpm run build
pnpm add -g serve
serve -s build
```
